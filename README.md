## Project Overview
The Expense Tracker is a responsive web application that helps users track their income and expenses.
Users can add, edit, delete, and categorize transactions while viewing real-time summaries and charts.

## Features
- Add income and expense transactions
- Categorize transactions
- Edit and delete existing transactions
- View total income, expenses, and balance
- Filter transactions by category
- Visualize expenses using a pie chart
- Data stored using browser LocalStorage
- Fully responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js

## How to Use
1. Open `index.html` in a web browser.
2. Fill in the transaction form (type, description, amount, category, date).
3. Click **Add Transaction**.
4. View totals and recent transactions.
5. Use category filter to view specific expenses.
6. Edit or delete transactions as needed.

## Storage
All data is saved locally in the browser using LocalStorage, so transactions persist after refresh.
